# Sorting Visualizer (Python + Tkinter)

## Overview

This project visualizes sorting algorithms using Python and Tkinter. It animates how different sorting algorithms work step-by-step on a random array of numbers.

## Features

- Generate random array
- Visualize Bubble Sort, Selection Sort, and Insertion Sort
- Adjustable speed of animation
- Highlights comparisons and swaps

## Tech Stack

- Python 3.x
- Tkinter GUI

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/sorting-visualizer.git
```

2. Navigate to the folder:

cd sorting-visualizer

3. Run the program:

python main.py
